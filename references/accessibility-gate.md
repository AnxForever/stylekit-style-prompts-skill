# Accessibility Gate

## Minimum Baseline

- Contrast target: WCAG AA (4.5:1 for normal text)
- Focus-visible states required for keyboard users
- Touch targets: at least 44x44px (or 24x24 minimum if constrained)
- Respect reduced motion preferences

## Prompt-Level Checks

- Mention interaction states (hover/active/focus/disabled)
- Mention accessibility terms (contrast, keyboard, WCAG, aria)
- Include mobile touch-target constraints
